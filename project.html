My Projects

From concept to code — a collection of academic, freelance, and personal projects showcasing my growth in full-stack development, mobile apps, and SEO.

01
Freelance · Responsive Website

AffordaLabs Diagnostic Plus

A fully responsive website built for AffordaLabs Diagnostic Plus, a medical diagnostic clinic in the Philippines. The goal was to establish a professional online presence, make clinic services and contact information easily accessible to patients, and boost organic visibility through on-page SEO strategies. I handled development, content structuring, UI/UX improvements, and performance enhancements from end to end.

Key Features

Responsive layout optimized for desktop, tablet, and mobile
Services section listing available diagnostic tests and packages
Contact page with clinic address, hours, and inquiry form
On-page SEO — meta tags, image alt texts, structured headings, keyword-rich content
Performance optimization including image compression and clean semantic HTML

Tools & Technologies

HTML5 CSS3 PHP MySQL On-Page SEO Responsive Design

02
Academic · Full-Stack Web App

Hotel Viñas Deluxe

A full-stack hotel booking and reservation system developed as an academic project. Guests can browse available rooms with photos and pricing, make reservations by selecting check-in/out dates, and receive booking confirmations. A dedicated admin panel allows hotel staff to manage room listings, monitor reservations, and update availability in real time — all backed by a PHP and MySQL database.

Key Features

Room browsing with photos, descriptions, amenities, and pricing
Online booking form with date picker and reservation confirmation
Admin dashboard to manage rooms, availability, and reservations
User authentication — register, login, session management
Fully responsive layout for all screen sizes

Tools & Technologies

PHP MySQL HTML5 CSS3 JavaScript XAMPP / Apache
Live Demo GitHub Repo
03
Academic · Mobile Application

Safe Sphere

Safe Sphere is a cross-platform mobile application built with Flutter and Dart, designed to promote personal safety and emergency awareness. The app gives users instant access to an SOS emergency feature, curated safety tips organized by category, and a clean intuitive interface built for fast usage in stressful situations. Developed as an academic capstone-level project, it runs natively on both Android and iOS devices.

Key Features

One-tap SOS emergency alert button with contact notification
Safety tips and awareness content categorized for quick access
Clean, distraction-free UI optimized for high-stress scenarios
Cross-platform support — Android & iOS via Flutter framework
User onboarding flow and personalized settings screen

Tools & Technologies

Flutter Dart Firebase Android Geolocation API
Mobile App GitHub Repo
04
SEO Optimization · WordPress

StarSeatAutoCover.com

An ongoing SEO optimization and web improvement project for StarSeatAutoCover, a Philippine-based car seat cover business. The project focuses on improving the site's organic search rankings, refining the landing page structure for better conversion rates, and implementing technical SEO best practices across the entire WordPress-powered website. Content is continuously updated to maintain relevance and ranking performance.

Key Features

Keyword research and integration into page titles, headings, and body copy
Optimized meta descriptions, OG tags, and canonical URLs
Image compression and alt-text optimization for faster load times
High-converting landing page layout with clear product sections and CTAs
Ongoing content updates to sustain search engine relevance

Tools & Technologies

WordPress HTML5 CSS3 On-Page SEO CRO Responsive Design
